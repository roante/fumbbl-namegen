Roante's fumbbl-namegen

How to set it up?

  1. Drag this bookmark fumbbl-namegen to your Bookmarks Bar.
    [Opera] [Firefox] [Chrome]
  2. When editing a team, copy and paste the URL into a new tab.
    This is important, the script won't work in the popup window except for Opera!
  3. Click on the bookmark to get a random name :-)

How to update?

If you used the previous version, the new version is updated automatically when you first use it after the update -- you don't have to do anything! Just wait a few seconds for the first time when you hit the bookmarklet since the new version has to be downloaded (takes about 5 seconds). After that the script will be cached and should work until the next update (if there will ever be).
