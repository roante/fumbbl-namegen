Roante's fumbbl-namegen

How to set it up?

  1. Drag this bookmark fumbbl-namegen to your Bookmarks Bar.
    [Opera] [Firefox] [Chrome]
  2. When editing a team, copy and paste the URL into a new tab.
    This is important, the script won't work in the popup window except for Opera!
  3. Click on the bookmark to get a random name :-)
